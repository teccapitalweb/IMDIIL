Proyecto web estático para IMDIIL.

Archivos principales:
- index.html
- css/styles.css
- js/script.js
- assets/img/

Para visualizarlo:
1. Descomprime la carpeta.
2. Abre index.html en tu navegador.

Incluye:
- Hero principal
- Contador animado
- Sección Nosotros
- Tarjetas de cursos con modal
- Galería visual con lightbox
- Opiniones en carrusel automático
- Contacto y botón flotante de WhatsApp
- Diseño responsive para celular
